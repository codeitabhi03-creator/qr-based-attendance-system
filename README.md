# qr-based-attendance-system
1. Start your Spring Boot app.

2. Open a browser and go to:
http://localhost:8080/class/CS101/qr
→ This shows the QR.

3. Scan the QR using your phone’s camera.

4. The phone opens a link with a small form.
→ Enter name and roll number.

5. Tap “Submit”.
→ It shows “Attendance Marked Successfully”.

6. Try scanning again from the same phone.
→ It shows “Already used from this device”.

7. To see records, open:
http://localhost:8080/attendance/CS101
→ Your entry appears there.
